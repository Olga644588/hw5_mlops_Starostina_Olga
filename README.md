# HW5 MLops Project

Проект по машинному обучению с использованием MLflow и DVC для воспроизводимого обучения модели классификации.


## Цель проекта


Цель проекта — создать воспроизводимый ML‑пайплайн с:
* версионированием данных через DVC;
* отслеживанием экспериментов через MLflow;
* фиксированными seed и версиями библиотек для гарантированной воспроизводимости результатов.

В качестве примера используется модель классификации Iris с логистической регрессией.

## Как запустить

1. Клонируйте репозиторий:
```bash
git clone <repo>
cd <repo>
2. Установите зависимости:
pip install -r requirements.txt
3. Загрузите данные через DVC:
dvc pull
4. Запустите пайплайн обработки и обучения:
dvc repro

Пайплайн состоит из двух основных стадий, описанных в dvc.yaml:
Стадия prepare (подготовка данных)
Вход:
сырые данные: data/raw/iris.csv;

скрипт обработки: src/prepare.py;

параметры из params.yaml (split_ratio, random_state).

Действия:

загрузка и базовая очистка данных;

разделение на обучающую и тестовую выборки с заданным соотношением;

сохранение обработанных данных в data/processed/.

Выход: обработанные данные (data/processed/train.csv, data/processed/test.csv).

Стадия train (обучение модели)
Вход:
обработанные данные: data/processed/ (результат стадии prepare);

скрипт обучения: src/train.py;

гиперпараметры из params.yaml (C, random_state);

модель: логистическая регрессия (LogisticRegression).

Действия:

загрузка данных и подготовка признаков;

обучение модели логистической регрессии;

предсказание и расчёт метрики качества (accuracy);

логирование в MLflow: параметров, метрики и артефакта модели;

сохранение обученной модели в model.pkl.

Выход:

обученная модель: model.pkl;

метрики в MLflow (accuracy);

артефакты (модель) в MLflow.

4. Смотреть UI MLflow. http://127.0.0.1:5000
