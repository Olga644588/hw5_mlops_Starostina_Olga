# HW5 MLops Project

Проект по машинному обучению с использованием MLflow и DVC для воспроизводимого обучения модели классификации.


## Цель проекта


Цель проекта — создать воспроизводимый ML‑пайплайн с:
* версионированием данных через DVC;
* отслеживанием экспериментов через MLflow;
* фиксированными seed и версиями библиотек для гарантированной воспроизводимости результатов.

В качестве примера используется модель классификации Iris с логистической регрессией.
